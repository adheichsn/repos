# Smarteye.id Creator

Platform Smarteye Creator berasal dari banyaknya keinginan konten creator untuk menampilkan objek secara mendetail dan menarik. 
Kebutuhan akan adanya platform Smarteye Creator ini membantu para konten creator atau user dalam membuat konten VR Tour dengan lebih mudah, tanpa memerlukan high-end device dan kemampuan programming yang rumit. 
Sehingga mereka dapat mewujudkan kreasi mereka dengan cepat tanpa keterbatasan alat. Smarteye creator ini juga menyediakan template berdasarkan segmentasi bisnis serta sarana membangun jaringan untuk para konten creator.
