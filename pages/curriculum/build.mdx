## How to Build

1. **Buka Build Settings :** Pergi ke File > Build Settings.
2. Untuk build ke oculus pastikan Platfiorm yang digunakan sudah Android.
3. Pastikan semua scene yang ingin di build sudah masuk dan diceklis.
4. Jika semua sudah selesai maka pilih build untuk melakukan prosesi build, jika masih ada yang perlu diubah bisa pilih Player Settings
