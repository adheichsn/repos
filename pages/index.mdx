#### **🤵Developer Role**

- Project Management
    
    Project management bertugas untuk menganalisis dan memantau seluruh kerjaan team agar dapat berjalan dengan sebagaimana mestinya. Project management juga akan dilibatkan dalam membangun flow journey pada virtual reality.
    
- Business Development
    
    Business development berperan untuk ditugaskan membuat pricing, serta melakukan analisis bisnis. bertanggung jawab dalam mengelola data serta melaporkan temuan data yang bisa diolah menjadi bisnis.
    
- Fullstack Developer
    
    Fullstack Developer berperan untuk ditugaskan membuat sebuah website sesuai journey yang telah dibuat, serta memahami konsep API karena tidak menutup kemungkinan akan berkolaboarasi dengan game developer.
    
- 3D Artist
    
    3D Artist berperan membuat konten evironment visual 3 Dimensi yang akan dimasukan kedalam teknologi VR yang canggih. serta mendevelop 3D asset, texturing dan animasi yang dibutuhkan oleh project.
    
- Game / Unity Engineer
    
    Unity Engineer akan mendevelop sebuah aplikasi Virtual Reality dengan menggunakan Unity. Tanggung jawab unity engineer sangat penting di project ini karena akan membuat sebuah logic mechanic dan menggabungkan semua asset dari 3D Artist serta UI/UX Designer.
    
- UI/UX Designer
    
    UI/UX Designer akan ditugaskan membuat design system serta UI design terhadap project virtual reality. UI/UX designer mempraktek kan prinsip prinsip UX serta dapat melakukan prototype.
    
- Video Animation
    
    Video Animarion memiliki peran membuat video animasi untuk keperluan opening di virtual reality. Video yang akan dibuat akan menyesuaikan project yang di assign.
