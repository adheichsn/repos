# Character Controller

- **Overview**: komponen siap pakai yang membantu developer secara cepat menyiapkan dan mengelola elemen dasar untuk interaksi VR, seperti kamera dan kontroler, serta memfasilitasi tindakan seperti mengambil dan menunjuk objek di Unity.
- **Steps to Create Features:**
    - Anda dapat mencari prefab yang telah disediakan oleh plugin XR Interaction Toolkit dengan memasukkan nama ‘XR Interaction Setup’ pada kolom pencarian. Kemudian drag prefab tersebut ke dalam scene.

        ![Untitled](https://prod-files-secure.s3.us-west-2.amazonaws.com/05cdfccc-adb3-469e-aac5-6dc11ecc1b49/c5f6c5b9-1aad-40c9-a4fb-948b5a5fe7c6/Untitled.png)

    - Untuk memudahkan pada saat debuging game, anda dapat menggunakan prefab ‘XR Device Simulator’ untuk menjalankan character controller menggunakan input dari PC.

        ![Untitled](https://prod-files-secure.s3.us-west-2.amazonaws.com/05cdfccc-adb3-469e-aac5-6dc11ecc1b49/34a0163a-e2b7-4264-a68e-ea9a1056ed11/Untitled.png)

    - Jika anda ingin menjalankan character controller pada saat play mode menggunakan oculus device, anda dapat menggunakan ‘oculus quest link’ untuk terhubung ke dalam unity. silahkan tonton tutorial berikut:
