# Popup Video on Smarteye Tour Creator

- **Overview**: Digunakan untuk menampilkan informasi berupa video, biasanya muncul saat user berinteraksi dengan elemen tertentu dalam game untuk memberi mereka penjelasan mengenai konten atau kondisi tertentu.


### Menambahkan popup teks

![popup image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/05cdfccc-adb3-469e-aac5-6dc11ecc1b49/ee9a68ba-e294-476c-aefe-92a522c47b46/popup_image.png)

Anda dapat menambahkan fungsi popup image melalui `**Menu Hotspot**`, kemudian pilih fungsi **`Video Popup`**.


### Mengatur konten yang akan ditampilkan


![Picture3.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/05cdfccc-adb3-469e-aac5-6dc11ecc1b49/6449a9b8-6760-48bb-bb91-cbdb8e2fe433/Picture3.png)

Tekan tombol `**Configure Pop-up**` untuk mengatur konten popup seperti menambahkan **`Konten Video, Judul dan Deskripsi video`**, masukkan video url berupa link youtube dan tambahkan slide dengan menekan `**tombol tambah**` untuk menampilkan konten video dalam beberapa slide. Jangan lupa untuk menyesuaikan posisi konten popup.
