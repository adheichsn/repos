# Popup Teks on Smarteye Tour Creator

- **Overview**: digunakan untuk menampilkan informasi berupa teks, biasanya muncul saat user berinteraksi dengan elemen tertentu dalam game untuk memberi mereka penjelasan mengenai konten atau kondisi tertentu.

### Menambahkan popup teks

![Picture2.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/05cdfccc-adb3-469e-aac5-6dc11ecc1b49/bc1beaf6-65e6-4094-9d66-25e9de07b5e5/Picture2.png)

Anda dapat menambahkan fungsi popup text melalui `**Menu Hotspot**`, kemudian pilih fungsi **`Text Popup`**.

### Mengatur konten yang akan ditampilkan

![Picture3.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/05cdfccc-adb3-469e-aac5-6dc11ecc1b49/6449a9b8-6760-48bb-bb91-cbdb8e2fe433/Picture3.png)

Tekan tombol `**Configure Pop-up**` untuk mengatur konten popup seperti menambahkan `**Judul dan Teks Deskripsi**`. Jangan lupa untuk menyesuaikan posisi konten popup.
