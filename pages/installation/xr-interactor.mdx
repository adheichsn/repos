# Panduan Instalasi XR Interaction Toolkit

## What is XR Interaction Toolkit

XR Interaction Toolkit di Unity adalah **sebuah alat yang membantu (Plugin)** kamu membuat dunia virtual dan augmented reality, yaitu dunia yang bisa kamu lihat melalui headset VR atau AR. Alat ini memberikan cara mudah untuk membuat karakter dan objek bisa berinteraksi dalam dunia virtual tersebut, seperti mengambil benda atau menggunakan controller untuk melakukan aksi.

## Installation and Set-up

![instal xri.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/05cdfccc-adb3-469e-aac5-6dc11ecc1b49/775101a8-2635-4c49-b6fd-e9caadfdedd9/instal_xri.png)

Untuk menginstal XR Interaction Toolkit di Unity, ikuti langkah-langkah berikut:

1. **Buka Unity Editor** dan buka proyek Anda.
2. **Buka Unity Package Manager**: Pergi ke **Window** > **Package Manager** di Unity Editor.
3. **Cari XR Interaction Toolkit**: Di Package Manager, Anda bisa mencari 'XR Interaction Toolkit' di kolom pencarian.
4. **Instal Package**: Setelah menemukan XR Interaction Toolkit, klik package tersebut dan tekan tombol 'Install' di panel detail.
5. **Tunggu Proses Instalasi**: Proses instalasi akan memakan waktu beberapa saat. Setelah selesai, XR Interaction Toolkit akan tersedia di dalam proyek Unity Anda.
6. **Verifikasi Instalasi**: Anda dapat memeriksa apakah toolkit telah terinstal dengan benar dengan mencari prefab atau komponen yang disediakan oleh toolkit di dalam Project Window.

Untuk menginstal XR Plugin Management dan mengonfigurasi plug-in providers untuk Oculus di Unity, ikuti langkah-langkah berikut:

1. **Buka Unity Editor** dan pilih proyek Anda.
2. Pergi ke **Edit** > **Project Settings**.
3. Dalam jendela Project Settings, cari dan pilih **XR Plugin Management**.
4. Di sini, Anda akan melihat opsi untuk menginstal XR Plugin Management jika belum terinstal. Klik **Install XR Plugin Management**.
5. Setelah terinstal, pilih platform target (misalnya, Windows atau Android untuk Oculus).
6. Centang kotak di samping Oculus untuk mengaktifkan Oculus sebagai XR Plug-in Provider untuk platform tersebut.
7. Pastikan Anda memiliki SDK Oculus yang diperlukan jika Anda mengembangkan untuk perangkat Oculus tertentu.
8. Setelah selesai, simpan pengaturan dan tutup jendela Project Settings.

Dengan mengikuti langkah-langkah ini, Anda akan mempersiapkan Unity untuk pengembangan dengan Oculus, memungkinkan Anda untuk membangun aplikasi VR yang kompatibel dengan perangkat Oculus.
