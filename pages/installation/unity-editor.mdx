# Panduan Instalasi Unity Editor Versi 2022.3.8f1

Silahkan download beberapa software berikut:

- Unity Hub: [***Link***](https://unity.com/download)
- Unity Editor Version 2022.3.8f1 : [***Link***](https://drive.google.com/file/d/1S22DBWj2TeHF_L7S83aqALwL94JCLI0_/view?usp=drive_link)

**Langkah 1 :** Extract unity 2022.3.8f1 ke folder yang anda inginkan.

![Untitled](https://prod-files-secure.s3.us-west-2.amazonaws.com/05cdfccc-adb3-469e-aac5-6dc11ecc1b49/81f1a161-bec5-4ef5-8518-dc2264d5f8ef/Untitled.png)

**Langkah 2 :** Lalu buka unity hub lalu pilih menu installs.

![Untitled](https://prod-files-secure.s3.us-west-2.amazonaws.com/05cdfccc-adb3-469e-aac5-6dc11ecc1b49/cefc943f-5e99-4dbb-b9e0-a338ee284536/Untitled.png)

**Langkah 3 :** Lalu pilih locate.

![Untitled](https://prod-files-secure.s3.us-west-2.amazonaws.com/05cdfccc-adb3-469e-aac5-6dc11ecc1b49/eb1231bf-37ca-4fc0-8625-287f08d3c372/Untitled.png)

**Langkah 4 :** Pilih ***unity.exe*** pada folder 2022.3.8f1 **>** 2022.3.8f1 **>** Editor. Lalu setelah itu select editor.

![Untitled](https://prod-files-secure.s3.us-west-2.amazonaws.com/05cdfccc-adb3-469e-aac5-6dc11ecc1b49/90b43019-0bdc-42a1-abff-6de4d4c7a5b6/Untitled.png)

**Langkah 5 :** Maka unity editor telah ter-install.
